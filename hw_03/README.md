# Анализ качества и очистка датасета мошеннических финансовых операций

### Команды для работы с Terraform в Makefile

```
make init
make validate
make plan
make apply
make destroy
make clean
```

Команда `make destroy` - уничтожает всё, кроме бакета и сервисного аккаунта.

### Описание

Бакет с паркетами доступен по адресу: `s3://otus-mlops-bucket-parquets-f02a83238337588f`.

Большой файл `main.tf` разделён по логическим ресурсам на несколько меньших файлов.

`.env.example` - пример того, как должен выглядеть файл `.env`, в который записывается информация о бакете в течение исполнения команды `make apply`.

`hw-03.ipynb` - ноутбук с демонстрацией.

`utils/data.py` - Python-скрипты для предобработки данных.

Скрипты в директории `infra/scripts/`:
- `upload_data_to_hdfs.sh` - копируем данные из бакета в HDFS;
- `upload_data_to_s3.sh` - копируем паркеты из HDFS в бакет.
